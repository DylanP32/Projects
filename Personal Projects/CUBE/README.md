Little mini project, Rotating Terminal Cube.

This project uses basic 3D math, including rotation matrices, projection, and back-face culling, to simulate real-time 3D rendering all in plain Python and NumPy.

Features:
- Real-time ASCII rendering in the terminal
- X/Y/Z rotation using rotation matrices
- Adjustable frame rate (default 60 FPS)
